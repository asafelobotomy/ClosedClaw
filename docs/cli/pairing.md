# `ClosedClaw pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
ClosedClaw pairing list whatsapp
ClosedClaw pairing approve whatsapp <code> --notify
```
