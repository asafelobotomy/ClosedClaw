# `ClosedClaw setup`

Initialize `~/.ClosedClaw/ClosedClaw.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
ClosedClaw setup
ClosedClaw setup --workspace ~/.ClosedClaw/workspace
```

To run the wizard via setup:

```bash
ClosedClaw setup --wizard
```
